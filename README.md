# 🚀 BestCode Control Panel (BCP)

Bem-vindo ao **BestCode Control Panel (BCP)**! O cérebro por detrás da magia do ecossistema.

## 🌟 O que é isto?

Em termos simples: é o teu **painel de controlo principal**.
Da mesma forma que usas o telemóvel para gerir as tuas apps, usas o BCP para gerir os teus sites, lojas online e infraestruturas (como servidores) de forma super simples!

Não precisas de perceber de códigos complexos para o usar, nós tratamos da parte chata e tu só tens que clicar em botões bonitos.

---

## ✨ Funcionalidades Incríveis

- 🌐 **Gestão de Sites Automática**: Cria, edita e configura projetos web em poucos segundos.
- 🎮 **Gestão de Servidores (Infraestrutura)**: O teu próprio sistema para gerir serviços complexos sem dores de cabeça.
- 🎨 **Visual Top & Simples**: Tudo desenhado para ser bonito, moderno, rápido e, acima de tudo, para "pessoas burras" (super intuitivo)!
- ⚡ **Super Rápido**: O teu painel não bloqueia, responde instantaneamente graças à tecnologia de ponta.

---

## 🛠️ Como Instalar? (Para Administradores)

É mais fácil que fazer um café! Só precisas de te ligar à tua máquina/servidor vazio e correr este comando mágico:

```bash
wget -qO- https://raw.githubusercontent.com/Crazed0/bestcode-cp/main/scripts/install.sh | bash
```

No final, o ecrã vai mostrar o teu link de acesso e as tuas credenciais iniciais! 🎉

---

## 🔄 Como Atualizar?

Se lançamos funcionalidades novas e queres atualizar o servidor, corre isto na consola do servidor:
```bash
cd /opt/bestcode-cp
git pull
npm install --prefix backend
pm2 restart all
```
Pronto, atualizado e sem tempo de inatividade!
